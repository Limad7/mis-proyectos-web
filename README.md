## 📝 Instrucciones de uso

1. **Clonar el repositorio:**
   ```bash
   git clone https://github.com/usuario/mis-proyectos-web.git
   cd mis-proyectos-web
   ```

2. **Para añadir un nuevo proyecto:**
   ```bash
   mkdir nombre-del-proyecto
   # añade archivos al nuevo directorio
   git add nombre-del-proyecto
   git commit -m "Añadir proyecto [nombre]"
   ```

3. **GitHub Pages:**
   - La web se desplegará en: https://usuario.github.io/mis-proyectos-web/[nombre-proyecto]/
